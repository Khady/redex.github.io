bs-node

This is a repo pulled out from BuckleScript's Node API (API here) so that it can be iterated on independently, in userland. It currently isn't stable! We'll gradually upstream the stable parts back into BuckleScript.

Installation

npm install --save buckletypes/bs-node

Then add bs-node to bs-dependencies in your bsconfig.json:

{
  ...
  "bs-dependencies": ["bs-node"]
}
